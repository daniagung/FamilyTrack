API PREPARE:

http://www.androidhive.info/2014/02/android-login-with-google-plus-account-1/ https://developers.google.com/+/mobile/android/sign-in?hl=en

1. Upload files from "Api Source Code" to your shared hosting/vps/server example:  api.domain.con or domain.com/api

2. Use "database_structure.sql" for your database structure.

3. In the curent folder open "class" folder and edit inform.class.php and change server,database, user and password to your mysql server details

USER INTERFACE PREPARE:

1. Upload files from "User Interface Source Code" to your shared hosting/vps/server (domain.com or domain.com/user)

2. Open the "config.php" file and edit variables for connection to your mysql server and other configs.

>

 

ANDROID STUDIO PREPARE:

1. Open app: Android Studio and select "Open an existing Android Studio project"

2. Change package name: https://www.youtube.com/watch?v=A-rITYZQj0A or http://stackoverflow.com/questions/25848046/how-to-change-package-name-in-android-studio

3. Open "app/java/packagename/utils/Config.java and change URL_SITE with your API URL and DEFAULT_SITE with User Interface URL

4. Don't forget to add your new package name to  https://console.developers.google.com/
